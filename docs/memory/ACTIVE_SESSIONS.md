# Active Sessions

Registry of developers currently working in this repository. Check this file at session start to avoid scope conflicts.

## Rules

1. **Register before starting**: Add a row with ACTIVE status before making edits.
2. **Deregister when done**: Change status to COMPLETED or remove your row.
3. **Check for overlaps**: If another developer's claimed files overlap yours, **stop and coordinate** before proceeding (use `COORDINATION.md` or an external channel).
4. **Stale entries**: Sessions with no commit activity for >24h may be marked STALE by any developer.
