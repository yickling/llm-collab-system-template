# Grunt Task Examples

| Example | File | Description | Task Types Used |

| Add Logging | [add-logging-to-module.md](add-logging-to-module.md) | Add structured logging to an existing auth module | INSERT, EDIT, VALIDATE |
| Rename and Update | [rename-and-update-imports.md](rename-and-update-imports.md) | Rename a file and update all import references | SEARCH_REPORT, RENAME, EDIT, VALIDATE |
| Create Config | [create-config-file.md](create-config-file.md) | Create a new config file, validate JSON, verify fields | RUN, CREATE, VALIDATE |
| Bulk Pattern Replace | [search-and-replace-pattern.md](search-and-replace-pattern.md) | Find a deprecated function, replace across files | SEARCH_REPORT, EDIT, VALIDATE |
